Elounda Family Font

Author: Valerio Brotto (Silverblur_type)
https://github.com/silverblurtype | vlrntype@gmail.com

Elounda is a geometric and minimalistic fonts constructed with a few geometric shapes with constant thickness. This font is suitable for graphic projects, logos and branding.
You can use this typeface for all personal and commercial works.
